
# Password Generator

A simple password generator app created with React JS



## Run Locally

Clone the project

```bash
  git clone https://github.com/paulgr98/password-generator.git
```

Go to the project directory

```bash
  cd password-generator
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
