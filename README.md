
# Password Generator

A simple password generator app created with React JS



## Features

- You can click on the password to copy it
- Dark mode
- Live previews
- Cross platform


## Run Locally

Clone the project

```bash
  git clone https://github.com/paulgr98/password-generator.git
```

Go to the project directory

```bash
  cd password-generator
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
